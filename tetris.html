My Own Version of Tetris

Tetris!

I have always loved intricate puzzles and problem solving. I played Tetris on my phone for a while, and then I started to wonder if I could make my own version. So, I did! I made this program using Java and while making it I learned how to use 2D Arrays and how to think like a problem solver.
